# simple-inventory

This is a simple self-hosted inventory system.

# How to install and run

To install bun go [look it up](https://bun.sh/docs/installation#installing).

To install dependencies:

```bash
bun install
```

To run:

```bash
bun start
```

# Files you don't have to worry about

- `tsconfig.json`
- `bun.lockb`
- `.gitignore`
- `node_modules/`
